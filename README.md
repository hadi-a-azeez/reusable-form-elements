# Reusable form elements with react

This project is to create reusabe react form components

## Available components

1. Input field
2. Button

## 1. Input field

[Usage]

`    <Input
        placeholder="Enter your name
        value={name}
        onChange={setName}
        style={{ width: "30%" }}
      />`
## 2. Button

[Usage]

     `<Button
        style={{ width: "30%", marginTop: "1rem" }}
        onClick={handleButtonClick}
     >
        Log in
      </Button>`
